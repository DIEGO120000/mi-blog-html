Ejemplo de estilo interno (CSS en el head)

En este ejemplo se muestra cómo aplicar estilos mediante CSS interno, es decir, escribiendo las reglas de estilo dentro de la etiqueta <style> en la sección <head> del documento HTML. Esta forma permite centralizar los estilos de una página y mantener separados el contenido (HTML) de la presentación (CSS), facilitando el mantenimiento y la organización.
