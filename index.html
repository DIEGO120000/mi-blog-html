Introducción

Este portafolio recopila las evidencias de los diferentes proyectos desarrollados durante el curso de Desarrollo Web, demostrando la aplicación de HTML5 y CSS3 para construir páginas estructuradas, con estilos personalizados, diseño adaptable y uso de animaciones modernas.

Reflexión Final sobre lo Aprendido

El desarrollo de estos proyectos permitió comprender la importancia de la estructura semántica del HTML y el poder del CSS3 para dar vida a las interfaces web. Además, reafirmó la relevancia del diseño responsivo y la organización del código para lograr páginas limpias, funcionales y visualmente agradables.
