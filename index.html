Reflexión Final sobre lo Aprendido

La realización de estos ejercicios fue esencial para entender que la clave del desarrollo web es la estructura y el significado semántico del código, no solo la apariencia visual. El mayor aprendizaje fue la importancia de organizar el contenido con etiquetas como <header>, <article> y <footer> para crear sitios accesibles y bien formados. Esto concluye la actividad.
